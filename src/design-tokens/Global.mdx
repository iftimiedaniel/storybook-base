# Global CSS Variables 

### How to use CSS Variables

The `var()` function is used to insert the value of a **CSS variable**.
The syntax of the `var()` function is as follows:

`var(--name, value)`

### Examples
These variables can be used inside stylesheets:

<Source 
    dark="true"
    type="code" 
    language="css" 
    code={`.myClassName { 
    width: var(--8-pt-numbers-7, 2rem); 
    height: var(--8-pt-numbers-7, 2rem); 
    background-color: var(--color-brands-primary-brand-color, #b93228);
}`} />
or inline
<Source 
    dark="true"
    type="code" 
    language="html" 
    code={`<div style="width: var(--8-pt-numbers-7, 2rem); 
    height: var(--8-pt-numbers-7, 2rem); 
    background-color: var(--color-brands-primary-brand-color, #b93228)">
</div>`} />
